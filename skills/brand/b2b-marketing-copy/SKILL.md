# B2B Marketing Copy — TCHUEKAM

Write marketing copy for untrained audiences who have never heard of TCHUEKAM. The user has explicitly stated: "pour une personne qui ne connaît pas c'est toujours compliqué la valeur doit être claire."

## Core Rule: Value First, Features Never

**WRONG** (user will reject this):
> "TchuekamUI est une plateforme d'automatisation visuelle et textuelle qui connecte des applications, des API et de la logique de manière totalement visuelle et orale."

**RIGHT** (user demands this):
> "TchuekamUI = un assistant qui travaille 24h/24 pour 25 000 FCFA/mois."

## The Value Formula

Frame every message as: **[Price] = [What it replaces] that [concrete benefit].**

- 25 000 FCFA/mois = un assistant qui ne dort jamais, ne râle pas, ne prend pas de congé
- 100 000 FCFA/mois = remplace un employé à 300 000 FCFA, travaille 24h/7
- 3 jours gratuits = vous testez, si ça marche on continue, sinon on arrête

## Output Formats

### Facebook Post (Long)
- Accroche: 3 lignes max contenant un chiffre choc ou une question douloureuse
- Corps: problème → solution → comparaison → CTA
- Sans jargon technique. Aucun mot comme "API", "no-code", "workflow", "intégration"

### Facebook Post (Court)
- 3-4 lignes max
- Value proposition + prix + CTA
- Idéal pour stories et carrousels

### WhatsApp Voice Note
- 55-60 secondes
- Ton posé, autoritaire, pas de musique
- Structure: Qui je suis → Ce que je fais → Prix → 3 jours gratuits → CTA ("tapez OUI")

### Landing Page
- Hero: une phrase valeur (pas un slogan vague)
- Problème: 2-3 lignes sur le temps perdu
- Solution: liste de tâches que TCHUEKAM fait
- Prix: tableau simple
- CTA: "Démarrer l'essai gratuit"

### Script Appel Commercial
- Question directe: "Vous perdez combien de temps par semaine à chercher des fichiers?"
- Proposition: "Si je vous disais qu'il existe un assistant pour 25 000 FCFA/mois?"
- Objection → réponse courte
- Clôture: "On installe maintenant ou plus tard?"

## What Never Goes In

- ❌ "Automatisation visuelle et textuelle" → ✅ "Vous parlez, il exécute"
- ❌ "Self-hosted, modifiable, open" → ✅ "C'est à vous, personne vous le coupe"
- ❌ "API, webhooks, déclencheurs" → ✅ Jamais, sous aucun prétexte
- ❌ "No-code, low-code, RPA" → ✅ Ces mots veulent rien dire pour le client
- ❌ "Plugins, modules, extensions" → ✅ "Il fait tout ça"
- ❌ "Développé par..., technologie basée sur..." → ✅ On dit ce qu'il FAIT, pas comment
- ❌ **Triple text repetition in headings** — Never write "AUTONOMOUS INTELLIGENCE AUTONOMOUS INTELLIGENCE AUTONOMOUS INTELLIGENCE" or any repeated heading text for visual effect. This is a CSS/design concern, not copy. Copy should be clean, single-instance text. If the designer wants 3x repetition for a glitch aesthetic, they handle it in CSS pseudo-elements, not in the copy layer.

## The "Switch Now" Format

For audiences already using a competitor (Make, Zapier, n8n):

- Start by naming their current tool (they self-identify)
- Ask uncomfortable questions: "Combien payez-vous? Êtes-vous bloqué par des limites?"
- Use a comparison table: Their tool vs TCHUEKAM on price, limits, sovereignty
- 3 days free trial as the closer
