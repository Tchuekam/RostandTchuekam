# The "Switch Now" Manifesto — Competitive Positioning Template

Written 28 May 2026 for TchuekamUI targeting Make/Zapier/n8n users.
Reusable pattern for any product that competes with an established tool.

## The 4 Critical Questions

Lead with these to expose pain in the incumbent:

1. "Combien payez-vous par mois pour exécuter des scénarios simples?"
2. "Avez-vous déjà été bloqué par une limite de tâches en pleine semaine?"
3. "Pouvez-vous modifier le code source? L'héberger sur votre propre machine?"
4. "Quand le serveur est down, qui appelez-vous?"
